
# DeepLearning_Server


usage : http://pwnable.co.kr:10004/api-docs


## Run Server


Clone the project

```bash
  git clone https://github.com/TaveConnectX/DeepLearning_Server.git
```

Go to the project directory

```bash
  cd DeepLearning_Server

```
Start the server

```bash
  nohup flask run -h 0.0.0.0 -p 10004 --debug &
```
